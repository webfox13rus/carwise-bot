# CarWise Bot — Telegram бот для учёта расходов на автомобиль

[![Python](https://img.shields.io/badge/Python-3.11-blue.svg)](https://www.python.org/)
[![Aiogram](https://img.shields.io/badge/Aiogram-3.10-green.svg)](https://docs.aiogram.dev/)
[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-15-blue.svg)](https://www.postgresql.org/)

**CarWise Bot** – ваш персональный авто‑помощник в Telegram. Бот помогает вести полный учёт расходов на автомобиль: заправки, обслуживание, страховки, напоминания о ТО и замене деталей, статистику, экспорт данных и AI‑советы.

---

## 🚀 Основные функции

- **Управление автомобилями** – добавление, редактирование, удаление. Поддержка более 50 марок (включая китайские).
- **Заправки** – учёт литров, суммы, пробега, типа топлива, фото чеков. Расчёт цены за литр и расхода.
- **Обслуживание** – ТО, мойка, ремонт, запчасти, шиномонтаж, жидкости. Возможность задать интервалы замены для деталей и жидкостей.
- **Страховки** – добавление полисов, напоминания за 7 и 3 дня до окончания.
- **Напоминания** – автоматические уведомления о ТО (по пробегу и времени), замене деталей/жидкостей, истечении страховки. Ежемесячные отчёты.
- **Статистика** – подробный отчёт по каждому автомобилю: общие расходы, средний расход, последние заправки, ближайшие замены, информация о страховке.
- **Экспорт данных** – выгрузка всех записей в CSV (для премиум‑пользователей). При большом объёме автоматически упаковывается в ZIP.
- **AI‑советы** – персональные рекомендации по обслуживанию от GigaChat (премиум‑функция).
- **Премиум‑подписка** – покупка через Telegram Stars. Доступны: неограниченное количество авто, экспорт, сравнение расходов, AI‑советы.
- **Админ‑панель** – управление пользователями, рассылка, блокировка, просмотр статистики, тестирование напоминаний.

---

## 🛠 Технологии

- **Python 3.11**
- **Aiogram 3.10** – асинхронный фреймворк для Telegram Bot API
- **SQLAlchemy 2.0** – ORM
- **PostgreSQL** – основная база данных
- **APScheduler** – планировщик для напоминаний
- **GigaChat** – API для AI‑советов
- **Railway** – хостинг (или любой PaaS с поддержкой Python и PostgreSQL)
